7.9.1.3. nisl.utils.check_niimgs¶

nisl.utils.check_niimgs(niimgs, accept_3d=False)¶

Check that an object is a list of niimg and load it if necessary

Parameters :

niimgs: (iterable of)* strings or objects :

If niimgs is an iterable, checks if data is really 4D. Then, considering that it is a list of niimg and load them one by one. If niimg is a string, consider it as a path to Nifti image and call nibabel.load on it. If it is an object, check if get_data and get_affine methods are present, raise an Exception otherwise.

accept_3d (boolean) :

If True, consider a 3D image as a 4D one with last dimension equals to 1.
Returns :

niimg: nibabel.Nifti1Image :

One 4D image. If 3D images were provided as input, this is the concatenation of all of them.

Notes

This function is the equivalent of check_niimg() for niimages with a session level.

Its application is idempotent.
